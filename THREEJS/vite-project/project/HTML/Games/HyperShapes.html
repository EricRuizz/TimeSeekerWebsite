INFORMATION

Description: Hyper Shapes is a frenetic minimalist Boss Rush where understanding the enemies attack patterns and weaknesses is key.

Engine: Unity
Team size: 3
Development Timeline: 10/09/2021 - 10/09/2022
Total Development Time: 1 year
Status: Multiplatform release on January 26th 2023 by JanduSoft
Context: Second-Year Bachelor's Project

MY CONTRIBUTION as a...

Game Designer

Designed the core gameplay and four of the ten bosses-the 1st, 4th, 9th, and 10th-, allowing me to introduce the player to the game and its mechanics, as well as build upon them using everything the player had experienced and learned.

Programmer

Fully implemented the Bosses I designed as well as the Boss Selection Menu and some other systems, such as my own automatic interpolator or the automatic room builder.

Artist

I was responsible for the overall visuals of the game in a very unexperienced team with no actual illustrators. By using simple shapes, solid colors, post processing and certaint types of motions in enemy movement, enemy transitions and particles, this stopped being an issue.

Composer

Produced my first ever soundtrack with the reduced amount of five tracks, one for the Main Menu, and four for the Bosses.
